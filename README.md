# 台指期 多時框看板 (TW Futures Multi-Timeframe Dashboard)

多時間框架技術指標看板，即時顯示台指期各時間框架的多空狀態。

## 功能

| 時框 | 本根 | KD | MA | VWAP | MC | SAR | ST | 分數 | 總結 |
|------|------|----|----|------|----|-----|----|------|------|
| 5分  | ▲/▼ | 多/空 | 多/空 | 多/空 | 多/空 | 多/空 | 多/空 | ±N | 偏多/中性/偏空 |
| 10分 | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| 30分 | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| 60分 | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| 日   | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| 週   | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| 月   | ... | ... | ... | ... | ... | ... | ... | ... | ... |

### 指標說明

| 指標 | 說明 | 參數 | 多空判斷 |
|------|------|------|----------|
| 本根 | K棒方向 | - | Close > Open = 多 |
| KD | 隨機指標 | 9,3,3 (台灣標準) | K > D = 多 |
| MA | 移動平均線 | 20期 | Close > MA = 多 |
| VWAP | 成交量加權均價 | 滾動20期 | Close > VWAP = 多 |
| MC | MACD | 12,26,9 | MACD > Signal = 多 |
| SAR | 拋物線SAR | AF=0.02, Max=0.2 | Close > SAR = 多 |
| ST | SuperTrend | ATR10, 乘數3 | 趨勢方向 = 多 |

- **分數** = 6個指標信號合計 (不含本根)
- **總結**: 分數 > 0 = 偏多, 分數 < 0 = 偏空, 分數 = 0 = 中性

### 資料來源

1. **CSV 檔案** (主要) - 支援 XQ 匯出格式
2. **Yahoo Finance** (輔助) - ^TWII 台灣加權指數即時資料
3. **TAIFEX API** (輔助) - 期交所報價 API

### 自動更新

- 每 5 分鐘自動刷新
- 可手動點擊更新按鈕

## 使用方式

### 直接執行 (Python)

```bash
pip install -r requirements.txt
python main.py
```

### 使用 EXE

1. 從 [Releases](../../releases) 下載 `TW_Futures_Dashboard.exe`
2. 將 EXE 放到 CSV 資料檔所在的資料夾
3. 雙擊執行

### 自行打包 EXE

```bash
build.bat
```

## CSV 資料格式

```
Date,Time,Open,High,Low,Close,Volume
20240329 , 150000 , 20286 , 20343 , 20285 , 20315 , 4596
```

支援的檔案名稱:
- `FIMTXN_1.TF_5m.csv` - 5分鐘K線
- `FIMTXN_1.TF_D.csv` - 日K線

- Python + tkinter
